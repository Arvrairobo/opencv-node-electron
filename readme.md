##install

- install opencv 2.4.*
- `npm install`
- `npm start`

##packages

- [electron](http://electron.atom.io/)
- [opencv](https://github.com/peterbraden/node-opencv)
- [socket.io](https://www.npmjs.com/package/socket.io)
